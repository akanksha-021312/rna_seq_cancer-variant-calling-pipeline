This folder contains scripts used in the variant calling workflow.

download_sra.sh – Download sequencing data from SRA  
fastqc.sh – Quality control of raw reads  
read_trimming.sh – Adapter trimming and filtering  
bwa_alignment.sh – Alignment to reference genome  
samtools_sort_index.sh – SAM to BAM conversion, sorting, and indexing  
call_variants.sh – Variant detection and filtering  
variant_annotation.sh – Annotation of identified variants
