# Dataset Description

The RNA-Seq data used in this variant calling project was obtained from the NCBI Sequence Read Archive (SRA).

Accession Number: SRR31860547

Sample Information:
- Sample: TPC-1 cell line
- Condition: sh-NC2
- Organism: Homo sapiens (Human)

This dataset is part of an RNA-Seq study investigating gene expression changes related to TP53 signaling in papillary thyroid carcinoma (PTC).

Data used in this project includes:
- Raw sequencing reads (FASTQ files)
- Aligned reads (BAM files)
- Variant call files (VCF)
- Annotated variant results

Raw sequencing files are not included in this repository due to large file sizes. The dataset can be accessed using the accession number provided above.
